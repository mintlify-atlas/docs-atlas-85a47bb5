## Overview

The Venue Management system allows administrators to manage all training locations (sedes) for Toluca Altas Montañas. Each venue has a name and a place (location description) that helps players identify the training site.

<Note>
  Venues are referenced by player registrations and schedules. Deleting a venue will cascade delete related schedules.
</Note>

## Component Location

The venue management interface is implemented in:

```
~/workspace/source/components/admin/sedesAdmin.tsx
```

It's accessed from the Admin Dashboard by clicking the "Sedes" navigation item.

## Data Model

### Venue Type

```typescript components/admin/sedesAdmin.tsx
type Venue = { 
  id: string;

  place: string | null 
};
```

### Database Table: `venues`

| Column | Type | Description |
|--------|------|-------------|
| `id` | UUID | Primary key (auto-generated) |
| `name` | TEXT | Venue name (e.g., "Río Blanco") |
| `place` | TEXT | Location description (e.g., "Río Blanco, Ver. - Estadio 7 de Enero") |

### Deleting a Venue

<Warning>
  Deleting a venue will cascade delete all schedules associated with that venue due to `ON DELETE CASCADE` foreign key constraints.
</Warning>

```tsx components/admin/sedesAdmin.tsx
const remove = async (v: Venue) => {
  const ok = confirm(
    "¿Eliminar esta sede? Si hay horarios ligados, se borrarán (on delete cascade)."
  );
  if (!ok) return;
  setBusy(true);
  setError(null);
  const { error } = await supabase.from("venues").delete().eq("id", v.id);
  if (error) setError(error.message);
  else await load();
  setBusy(false);
};
```

## Data Loading

Venues are loaded from Supabase on component mount:

```tsx components/admin/sedesAdmin.tsx
const load = async () => {
  setLoading(true);
  setError(null);
  const { data, error } = await supabase
    .from("venues")
    .select("id, name, place")
    .order("name");
  if (error) {
    setError(error.message);
    setVenues([]);
  } else {
    setVenues((data ?? []) as Venue[]);
  }
  setLoading(false);
};

useEffect(() => {
  load();
}, []);
```

### Create

```tsx components/admin/sedesAdmin.tsx

  name: form.name.trim(),
  place: form.place.trim(),
});

  setOpen(false);
  await load();
```

```tsx components/admin/sedesAdmin.tsx

  })
  .eq("id", editing.id);

if (error) setError(error.message);
else {

}
```
