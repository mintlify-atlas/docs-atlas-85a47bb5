## Overview

The Schedule Management system (Horarios Admin) allows administrators to define when each category trains at each venue. Schedules are organized by weekday, time range, category, and venue.

<Note>
  Schedules are venue-specific. Administrators select a venue first, then configure schedules for different categories at that location.
</Note>

## Component Location

The schedule management interface is implemented in:

```
~/workspace/source/components/admin/horariosAdmin.tsx
```

It's accessed from the Admin Dashboard by clicking the "Horarios" navigation item.

## Data Model

### Schedule Type

```typescript components/admin/horariosAdmin.tsx
type Schedule = {
  id: string;
  category_id: string;
  venue_id: string;
  weekday: number; // 1-7 (Monday-Sunday)
  start_time: string; // "16:00:00"
  end_time: string;
  is_optional: boolean;
  note: string | null;
};
```

### Database Table: `schedules`

| Column | Type | Description |
|--------|------|-------------|
| `id` | UUID | Primary key |
| `category_id` | UUID | Foreign key to categories table |
| `venue_id` | UUID | Foreign key to venues table |
| `weekday` | INTEGER | Day of week (1=Monday, 7=Sunday) |
| `start_time` | TIME | Training start time |
| `end_time` | TIME | Training end time |
| `is_optional` | BOOLEAN | Whether this session is optional |
| `note` | TEXT | Optional notes about the session |

## Time Formatting

Times are converted between input format (HH:mm) and database format (HH:mm:ss):

```typescript components/admin/horariosAdmin.tsx
function tToInput(t: string) {
  return (t ?? "").slice(0, 5); // "16:00:00" → "16:00"
}

function inputToDb(t: string) {
  return t.length === 5 ? `${t}:00` : t; // "16:00" → "16:00:00"
}
```

## Data Loading

The component loads three datasets in parallel:

```typescript components/admin/horariosAdmin.tsx
const load = async () => {
  setLoading(true);
  setError(null);

  const [vRes, cRes, sRes] = await Promise.all([
    supabase.from("venues").select("id, name, place").order("name"),
    supabase
      .from("categories")
      .select("id, name, year_from, year_to, sort_order")
      .order("sort_order", { ascending: true }),
    supabase
      .from("schedules")
      .select(
        "id, category_id, venue_id, weekday, start_time, end_time, is_optional, note"
      )
      .order("weekday", { ascending: true })
      .order("start_time", { ascending: true }),
  ]);

  // Process results...
};
```

## Schedule Grouping

Schedules are grouped by category for organized display:

```typescript components/admin/horariosAdmin.tsx
const grouped = useMemo(() => {
  const list = schedules.filter((s) => s.venue_id === venueSel);

  // Group by category
  const map = new Map<string, Schedule[]>();
  for (const r of list) {
    const arr = map.get(r.category_id) ?? [];
    arr.push(r);
    map.set(r.category_id, arr);
  }

  const entries = Array.from(map.entries()).sort((a, b) => {
    const sa = catById.get(a[0])?.sort_order ?? 9999;
    const sb = catById.get(b[0])?.sort_order ?? 9999;
    return sa - sb;
  });

  return entries.map(([category_id, rows]) => ({
    category_id,
    rows: rows.slice().sort((x, y) => x.weekday - y.weekday),
  }));
}, [schedules, venueSel, catById]);
```

## CRUD Operations

### Create

```typescript components/admin/horariosAdmin.tsx
const payload = {
  venue_id: venueSel,
  category_id: form.category_id,
  weekday: Number(form.weekday),
  start_time: inputToDb(form.start_time),
  end_time: inputToDb(form.end_time),
  is_optional: !!form.is_optional,
  note: form.note.trim() ? form.note.trim() : null,
};

const { error } = await supabase.from("schedules").insert(payload);
```

### Update

```typescript components/admin/horariosAdmin.tsx
const { error } = await supabase
  .from("schedules")
  .update(payload)
  .eq("id", editing.id);
```

### Delete

```typescript components/admin/horariosAdmin.tsx
const remove = async (r: Schedule) => {
  const ok = confirm("¿Eliminar este día/horario? No se puede deshacer.");
  if (!ok) return;
  setBusy(true);
  setError(null);
  const { error } = await supabase.from("schedules").delete().eq("id", r.id);
  if (error) setError(error.message);
  else await load();
  setBusy(false);
};
```
